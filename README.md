## 🕹️ HACKLAB — Learn Ethical Hacking by Playing

A mission-based game that teaches ethical hacking, network security and
penetration testing through interactive, fully-simulated challenges. Lives in
[`/hacklab`](hacklab/). Open `hacklab/index.html` to play, or once the site is on
GitHub Pages: `https://<your-username>.github.io/<repo>/hacklab/`.

See [hacklab/README.md](hacklab/README.md) for details.
